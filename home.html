웹페이지 기초

웹페이지는 HTML, CSS, JavaScript를 사용하여 만들어집니다.

HTML

HTML은 웹페이지의 구조를 정의하는 마크업 언어입니다.

기본적인 HTML 태그

  • <html></html>: HTML 문서의 시작과 끝을 나타냅니다.
  • <head></head>: 문서의 메타 정보를 포함합니다.
  • <title></title>: 문서의 제목을 나타냅니다.
  • <body></body>: 문서의 본문 내용을 포함합니다.
  • <h1></h1>~<h6></h6>: 제목을 나타냅니다.
  • <p></p>: 단락을 나타냅니다.
  • <ul></ul>: 목록을 나타냅니다.
  • <li></li>: 목록 항목을 나타냅니다.

CSS

CSS는 웹페이지의 디자인을 정의하는 스타일시트 언어입니다.

JavaScript

JavaScript는 웹페이지에 동적인 기능을 추가하는 프로그래밍 언어입니다.
